# TwitchChatOverlay
Simple chat overlay

To start use
```sh
java -jar TwitchChatOverlay-0.0.1.jar "#twitch_channel_name"
```
[Download java binary](https://github.com/uaBArt/PingJa/blob/master/TwitchChatOverlay-0.0.1.jar)
